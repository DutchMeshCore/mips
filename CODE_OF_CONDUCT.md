# Code of Conduct

This project adopts the Contributor Covenant Code of Conduct 2.1.

For the full text, see:
[https://www.contributor-covenant.org/version/2/1/code_of_conduct/](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)

## Scope

This applies to issues, pull requests, discussions, and other community spaces connected to this repository.

## Reporting

Report conduct issues privately to maintainers using agreed private channels.
